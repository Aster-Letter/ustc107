# 数学与统计用例

本节面向数学、统计、概率、优化、数值计算等方向的课程与实验。

## 适合放在这里的内容

- 大规模矩阵计算、数值线性代数与优化实验；
- Monte Carlo 模拟、随机过程与统计推断；
- 使用 Python、R、Julia、MATLAB 或 C/C++ 的批处理任务；
- 需要并行 CPU、较大内存或 GPU 加速的课程作业。

## 建议首批用例

- 使用 Python/NumPy/SciPy 提交数值计算作业；
- 使用 R 批量运行统计模拟；
- 使用 Julia 运行优化或微分方程实验；
- 使用 GPU 加速矩阵运算或自动微分任务。

数学与统计页面应重点说明随机种子、参数扫描、输出结果组织和复现实验，而不是只给出单次运行命令。
