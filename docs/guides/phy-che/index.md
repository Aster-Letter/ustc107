# 物理与化学用例

本节面向物理、化学、材料、生命科学等需要仿真、建模或数据处理的课程与实验。

## 适合放在这里的内容

- 分子动力学、量子化学、材料模拟等计算任务；
- 大规模实验数据处理与可视化；
- 使用课程指定软件或开源模拟工具的批处理任务；
- 需要长时间 CPU 计算、GPU 加速或较大存储空间的实验。

## 建议首批用例

- 提交一个 CPU 多核仿真任务；
- 提交一个 GPU 加速模拟任务；
- 批量处理实验数据并保存图表；
- 从图形界面迁移到作业脚本运行。
