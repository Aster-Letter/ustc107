# 开发日志

本文记录文档站点各版本的主要变化、已知不足和后续计划。

## v0.1 测试版

发布日期：2026-05-21

主要更新：

- 建立文档站点基础结构，覆盖首页、快速开始、平台介绍、基础使用、分专业与需求指引、开发说明等板块。
- 完成 GUI 使用、命令行使用、CLI 命令索引、环境配置、文件与数据、提交任务、Slurm 速查、常见问题等核心页面。
- 新增深度学习作业训练任务指南，并为数学与统计、物理与化学、其他通用计算场景保留扩展入口。
- 补充平台正式入口、默认资源示例、QOS、交互式任务、Web Shell 复制粘贴、GPU 检查、上传校验等说明。
- 加入部分示例配图，用于说明登录页、仪表盘、应用、文件管理、作业提交、日志预览和 Web Shell 操作。
- 新增 README、写作指南和开发日志，说明本地预览、构建检查、截图脱敏、动态事实核验和协作流程。

已知不足：

- 部分资源策略、GPU 型号、节点状态、QOS 权限和默认配额仍具有即时性，需要定期复核。
- 创建 VS Code 应用、交互式进入计算节点、资源申请、作业失败排查等流程还需要更多清晰截图。
- FAQ 中部分条目仍需要按“已验证”“待验证”“可能过期”继续分类。
- 当前版本主要面向现平台，后续平台迁移或架构重构后，页面入口和操作流程可能需要调整。

下一步建议：

- 由平台维护者确认正式入口、默认配额、Students 分区/QOS、SSH 状态和资源申请流程。
- 补充创建 VS Code 应用、交互式 `srun`、资源申请、作业失败日志等截图。
- 将 FAQ 中的高频问题按“已验证”“待验证”“可能过期”继续分类。
- 根据实际课程案例扩展数学统计、物理化学和通用计算场景指南。
